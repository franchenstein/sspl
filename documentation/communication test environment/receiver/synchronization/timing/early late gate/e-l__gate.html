e-l_gate.vi

 input complex waveform

specifies the continuous-time samples of the modulated complex baseband waveform data. Wire the output complex waveform returned from the MT Matched Filter VI to this input.

NOTE: For proper operation of this VI, the following requirements must be met:

--The sampling rate of the input complex waveform must be an even integer multiple of the expected symbol rate. Use the MT Fractional Resample VI to adjust the input complex waveform sampling rate if neccessary.

--An appropriate matched filter must be previously applied to the input complex waveform. Use the MT Matched Filter VI to apply the matched filter.

    		t0

    specifies the trigger (start) time of the Y array.

    		dt

    specifies time interval between data points in the Y array.

    		Y

    specifies the matched filtered complex-valued signal-only baseband modulated waveform. The real and imaginary parts of this complex data array correspond to the in-phase (I) and quadrature-phase (Q) data, respectively.

      		Numeric

error in (no error)

alignment offset (samples)

returns the location of the ideal symbol timing instant in samples relative to the beginning of the input complex waveform. This offset was applied to the output complex waveform. Multiply this value by the input complex waveform dt value to calculate the location of the ideal symbol timing instant in seconds.

aligned complex waveform

returns the symbol-time-aligned IQ baseband waveform. Wire this output to the MT Decimate Oversampled Waveform VI followed by the MT Map Symbols to Bits VI to obtain the recovered bits corresponding to the modulated waveform.

    		t0

    outputs the trigger (start) time of the Y array.

    		dt

    outputs time interval between data points in the Y array. The reciprocal of this value is equal to the system sampling rate.

    		Y

    outputs the complex-valued signal-only baseband modulated waveform. The real and imaginary parts of this complex data array correspond to the in-phase (I) and quadrature-phase (Q) data, respectively.

      		Numeric
